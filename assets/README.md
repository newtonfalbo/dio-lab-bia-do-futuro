# 🎨 Assets

Esta pasta reúne todos os recursos visuais utilizados pelo projeto **Falbo AI**.

## Estrutura

| Arquivo | Descrição |
|---------|-----------|
| `banner-falbo-ai.png` | Banner utilizado no README do projeto. |
| `arquitetura-falbo-ai.png` | Diagrama da arquitetura do agente. |
| `fluxo-agente.png` | Fluxograma do funcionamento do agente e da integração com a base de conhecimento. |
| `logo-falbo-ai.png` | Logotipo do projeto. |
| `demo.gif` | Demonstração da aplicação em funcionamento. |
| `screenshots/` | Capturas de tela da aplicação e dos resultados obtidos. |

---

## Objetivo

Os recursos desta pasta são utilizados para:

- Documentação técnica;
- README do projeto;
- Apresentação (Pitch);
- Demonstração da arquitetura;
- Ilustração do funcionamento do agente.

---

## Convenções

- Utilize nomes de arquivos em letras minúsculas.
- Separe palavras com hífen (`-`).
- Prefira imagens em PNG.
- Para animações utilize GIF.
- Evite arquivos muito grandes.

Exemplo:

```
assets/
│
├── banner-falbo-ai.png
├── arquitetura-falbo-ai.png
├── fluxo-agente.png
├── logo-falbo-ai.png
├── demo.gif
└── screenshots/
    ├── tela-inicial.png
    ├── consulta-produto.png
    └── resultado-analise.png
```
